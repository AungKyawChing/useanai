List of HTML Attributes

Ordered List (ol)

  1. id - Specifies a unique id for an HTMl element
  2. class - Specifies one or more class name for an HTML element (used to style elements with CSS)
  3. href - Specifies the URl of the page the link goes to
  4. src - Specifies the URl of the image to be displayed
  5. alt - Specifies an alternative text for images(in case the image can't be displayed)

Unordered List (ul)

  • title - Specifies extra information about an element (displayed as a tooltip)
  • style - Specifies an inline CSS style for an HTML element
  • width - Specifies the width of an element
  • height - Specifies the height of an element
